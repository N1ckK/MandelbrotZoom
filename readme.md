
![MandelbrotMain](https://i.imgur.com/CubP8Ww.png)

### Features:
- Random coloring on every launch.
- Zoom up to 10^12

### Versions:
```Mandelbrot45.exe```

*For for better performance:*
- Window Dimensions: 500x400
- Fractal Depth: 45


```Mandelbrot90.exe```

*Recommended:*
- Window Dimensions: 800x600
- Fractal Depth: 90

![Mandelbrot](https://i.imgur.com/NFLXS5O.png)

### Controls:

``W - Move Camera UP``
``A - Move Camera LEFT``
``S - Move Camera DOWN``
``D - Move Camera RIGHT``
``Q - Increase Zoom``
``E - Decrease Zoom``

![Mandelbrot](https://i.imgur.com/tkWkLZo.png)

### Libraries Used:
- SDL2 (2.0.9)
